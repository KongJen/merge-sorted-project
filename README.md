# merge-sorted-project

this is test merge sort by using jest to test in typescript

## Pre Require 
- Node.js >= 14
- npm >= 6

## Installation
- Clone repository:
  ```
  git clone https://github.com/KongJen/merge-sorted-project.git
  cd merge-sorted-project
  ```

- install dependencies:
  ```
  npm install
  ```

## Running Unit Tests
  ```
  npm test
  ```
